# Sports Broadcast Verifier

A free sports broadcast verification app built with Python and Streamlit.

## Deploy Free on Streamlit Community Cloud

1. Upload these files to your GitHub repo
2. Go to share.streamlit.io and sign in with GitHub
3. Click New app, select your repo, set main file to app.py
4. Click Deploy

## Features

- Import schedule from Google Sheets or CSV upload
- Pre-event cross-check and day-after verification
- Actual start time and actual end time columns
- National channel, local home RSN, local away RSN, and streaming columns
- Color-coded verdicts: AIRED_AS_SCHEDULED, DELAYED, POSTPONED, CANCELED, NETWORK_CHANGED, UNVERIFIED
- One-click Google fallback search link on every row
- Run history saved to database
- CSV export
- Add new sports/leagues from the sidebar with no code changes
