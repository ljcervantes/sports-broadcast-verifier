# Sports Broadcast Verifier (Streamlit)

A free, easy-to-deploy sports broadcast verification app built with Python and Streamlit.

## Features

- Import schedule from Google Sheets (paste a shareable link) or CSV upload
- Pre-event cross-check and day-after verification
- Actual start time and actual end time columns (via Google Search)
- National channel, local home RSN, local away RSN, and streaming columns
- Color-coded verdicts: AIRED_AS_SCHEDULED, DELAYED, POSTPONED, CANCELED, NETWORK_CHANGED, UNVERIFIED
- One-click Google fallback search link on every row
- Run history saved to SQLite database
- CSV export of any run
- Add new sports/leagues from the sidebar — no code changes needed
- 200+ local station call signs built in (KMCC, SPECSN, KUNS, NESN, YES, Bally Sports, etc.)

## Deploy Free on Streamlit Community Cloud

1. Fork or upload this repo to your GitHub account
2. Go to [share.streamlit.io](https://share.streamlit.io) and sign in with GitHub
3. Click **New app** → select your repo → set main file to `app.py`
4. Click **Deploy** — your app is live in about 2 minutes
5. Add your SerpApi key in the app sidebar (Settings section)

## Local Development

```bash
pip install -r requirements.txt
streamlit run app.py
```

## SerpApi Key (for Google Search lookups)

- Free tier: 100 searches/month at [serpapi.com](https://serpapi.com)
- Paid: $50/month for 5,000 searches
- Without a key: the app still works using the built-in local station library

## Verdict Reference

| Verdict | Color | Meaning |
|---|---|---|
| AIRED_AS_SCHEDULED | Green | Aired on correct channel at scheduled time |
| MATCH | Green | Pre-event sources agree |
| DELAYED | Orange | Rain delay or other delay |
| NETWORK_CHANGED | Orange | Moved to different channel |
| MISMATCH | Orange | Sources disagree |
| POSTPONED | Red | Pushed to future date |
| CANCELED | Red | Called off entirely |
| UNCONFIRMED | Yellow | Data found but not fully confirmed |
| UNVERIFIED | Yellow | No automated source could confirm |
